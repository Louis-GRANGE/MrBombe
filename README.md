# Informations
Ce projet est créé pour apprendre à utiliser le moteur de jeu Unreal Engine dans l'une de ces derniéres versions la 4.26.   
L'objectif est de créer un jeu ou une application avec aucune contraintes sauf l'utilisation du moteur de jeu.

# Description
Mr Bombe est un jeu multijoueur en local. Chaque joueur étant dans un premier temps seul et devra survivre jusqu’à la fin, pour cela il devra poser des bombes pour casser des blocs et récupérer des bonus.    
Pour gagner il doit être le dernier en vie et tuer les autres joueurs à l’aide de ses bombes. Au départ, le joueur ne peut poser qu’une bombe à la fois.  

Lorsque le joueur casse un bloc il pourra probablement obtenir un bonus tel que:

* Vitesse amélioré
* +1 Bombe posable
* augmentation de la longueur des explosions de bombes
* ... 

La carte du jeu est générée procéduralement, dont certains blocs sont indestructibles.
